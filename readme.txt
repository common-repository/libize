Wp Libize est une extension pour lier sa boutique à l'application Libize.

== Description ==

Wp Libize est le plugin de connexion de l'application Libize à son site de e-commerce WooCommerce.

Libize est un extracteur automatique de fiches produits. Un outil indispensable quand on gère un site de e-commerce.

Comment ça marche ? En quelques clics, Libize récupère les infos e-business dont vous avez besoin, et les importe automatiquement sur votre propre site.

== Installation ==

= Depuis son tableau de bord WordPress =

1. Naviguer vers Extensions -> Ajouter.
2. Cliquer sur le bouton "Téléverser une extension" à côté du titre "Ajouter des extensions".
3. Cliquer sur Choisir un fichier et sélectionner "wp-libize.zip" et l'activer.

= Activer la liaison =

1. Dans l'onglet réglages, sélectionner Libize.
2. Dans la section Configuration Wp Libize, cliquer sur "Lier votre boutique avec Libize"
3. Se connecter à Libize puis accepter la liaison.
